SimpleIDGChatWidgetListener

open class SimpleIDGChatWidgetListener : IDGChatWidgetListener

A simple implementation of IDGChatWidgetListener that provides empty default implementations for all Chat SDK callbacks. This class can be extended to create custom listeners that only need to override specific callbacks.
